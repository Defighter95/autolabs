УСТАНОВКА

Чтобы установить программу AutoLabs на свой ПК, запустите установочный файл setup.exe в папке с продуктом и следуйте подсказкам установщика
Для корректной работы приложения на Вашем ПК должна быть устанолвена СУБД MS Access версии 2003 или более новая.

=======================================================

НАСТРОЙКА

AutoLabs.Server

1. После завершения установки, запустите программу, используя ярлык на рабочем столе, либо файл AutoLabs - Server.exe в папке с ПО. 
2. Выполните вход, пароль - 1111. После первого входа настоятельно рекомендуется изменить пароль, используя пункт меню Опции -> Сменить пароль.

AutoLabs.Client

1. После завершения установки, запустите программу, используя ярлык на рабочем столе, либо файл AutoLabs - Client.exe в папке с ПО. 
2. При первом запуске программа попросит указать начальные настройки: каталог с установленным приложением - сервером, настройки строки поключения (кнопка Build -> поставщик данных - Microsoft Jet 4.0 OLE DB -> подключаемая база данных находится в папке с сервером по имени db.mdb), и имя вашего ПК.
3. При первом входе, студенту необходимо установить пароль (пароль состоит из 4-х символов, допускаются исключительно цифры)


=======================================================

ПОДСКАЗКИ ПО РАБОТЕ И ПОЛЕЗНАЯ ИНФОРМАЦИЯ

— Не забывайте периодически создавать резервные копии базы данных (пункт меню Опции -> Создать резервную копию)  
- При подозрении, что Ваш пароль узнали другие лица, рекомендуется немедленно сменить его (пункт меню Опции -> Сменить пароль)
- Старайтесь не использовать в качестве исходников для лабораторных работ файлы слишком большого объёма (> 500 MB), так как программа копирует исходные файлы в свой каталог ресурсов и может быстро занять всё свободное место на Вашем диске
- Если сообщения об ошибках появляются слишком часто, программа работает медленно и/или некорректно, либо у Вас есть другие замечания или пожелания - обязательно напишите нам об этом, Вы внесёте неоценимый вклад в развитие проекта (наш email: iliakobus@yandex.ru) 
- Программа поддерживает ограниченый набор форматов файлов для исходников лабораторных работ (.mdb, .accdb, .doc, .docx, .ppt, .xls и пр. форматы документов MS Office), работоспособность файлов других форматов не гарантируется

FAQ

Вопрос: Не удаётся настроить подключение к Базе Данных, либо поключённая БД работает некорректно.
Ответ: Убедитесь, что Вы подключаете правильную БД (база данных приложения находится в одной папке с AutoLabs. Server и имеет имя db.mdb), а также на Вашем компютере установлена СУБД MS Access ерсии 2003 или более новая.   
.
Вопрос: При запуске ЛР на выполнение возникает ошибка работы со сторонним приложением и программа закрывается.
Ответ: Формат файла исходников для ЛР должен быть поддерживаем программой (форматы файлов MS Office), а также приложение, которое выполняет открытие фалйа должно быть правильно установлено и настроено. 

Вопрос: При добавлении ЛР возникает ошибка "Не найден файл Источник".
Ответ: Проверьте правильность имени файла, файл должен быть закрыт и не занят другим процессом для успешного копирования в каталог ресурсов приложения.

Вопрос: Я забыл свой пароль и не могу войти в приложение, мой пароль был создан другим пользователем перед моим первым входом, либо был украден и изменён.
Ответ: Обратитесь к администратору сети для восстановления своего доступа.
